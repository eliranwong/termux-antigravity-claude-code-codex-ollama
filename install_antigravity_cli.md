# Install Antigravity CLI

1. Install Antigravity CLI on Termux directly:

Reference: https://github.com/wallentx/antigravity-cli-termux

To install, run:

```
curl -fsSL https://raw.githubusercontent.com/wallentx/antigravity-cli-termux/dev/install.sh | bash
```

To Update, DO NOT run `agy update`, install re-run the install script:

```
curl -fsSL https://raw.githubusercontent.com/wallentx/antigravity-cli-termux/dev/install.sh | bash
```

2. Install Ollama inside Ubuntu environment

First, login Unbuntu Environment

For details, read [install_ubuntu.md](install_ubuntu.md).

ubuntu
